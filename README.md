## Modelos

Este modelo de IA es del tipo de `Image Classification`, con el cual podemos predicir con una imagen el tipo de vegetal que es. La lista de los vegetales que es capaz de predicir es la siguiente:

- Bean
- Bitter_Gourd
- Bottle_Gourd
- Brinjal
- Broccoli
- Cabbage
- Capsicum
- Carrot
- Cauliflower
- Cucumber
- Papaya
- Potato
- Pumpkin
- Radish
- Tomato
